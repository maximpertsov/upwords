upwords
=======

Upwords Boardgame

===================================================================
  Rules taken from: http://boardgamegeek.com/boardgame/1515/upwords

  10 x 10 version has the following letters:

  J, Qu, V, X, Z, - 1 of Each
  K, W, Y - 2 of Each
  B, F, G, H, P, - 3 of Each
  C, - 4 of Each
  D, L, M, N, R, T, U, - 5 of Each
  S, - 6 of Each
  A, I, O, - 7 of Each
  E, - 8 of Each

===================================================================
